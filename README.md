# Тестовое задание Performance Lab

Данный репозиторий содержит решения 4 алгоритмических задач на языке Python, выполненные в рамках тестового задания для Performance Lab.

## Структура проекта

```text
.
├── task1/              # Задание 1: Круговой массив
│   └── task1.py
├── task2/              # Задание 2: Положение точек относительно эллипса
│   ├── task2.py
│   ├── circle.txt      # Пример файла с параметрами эллипса
│   └── dot.txt         # Пример файла с координатами точек
├── task3/              # Задание 3: Формирование JSON отчета
│   ├── task3.py
│   ├── values.json     # Пример файла с результатами тестов
│   ├── tests.json      # Пример файла со структурой тестов
│   └── report.json     # Генерируемый файл отчета
├── task4/              # Задание 4: Минимальное количество ходов
│   ├── task4.py
│   └── nums.txt        # Пример файла с массивом чисел
├── author.txt          # Информация об авторе
└── README.md           # Описание проекта
